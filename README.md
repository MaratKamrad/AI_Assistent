## 🔍 Обзор проекта

**ИИ-ассистент для компании по установке окон** — это интеллектуальный чат-бот, предназначенный для автоматизации первичного общения с клиентами, консультаций по продукции и услугам, а также организации встреч с менеджерами.

Проект реализован на базе современных нейросетевых технологий и позволяет:
- Снижать нагрузку на операторов кол-центра
- Обеспечивать мгновенные ответы 24/7
- Повышать конверсию за счёт быстрого вовлечения клиента
- Автоматизировать рутинные задачи записи и напоминаний

## ⚙️ Функциональные возможности

### 1. 🤖 Осмысленное ведение диалога
- Понимание естественного языка (NLU) на русском языке
- Поддержка многоходовых диалогов с сохранением контекста
- Распознавание намерений (intents): «заказать замер», «узнать цену», «гарантия» и др.
- Адаптивный тон общения: от формального до дружелюбного

### 2. 📚 Ответы на вопросы по базе знаний
- Мгновенный поиск информации в структурированной базе:
  - Типы окон (ПВХ, дерево, алюминий)
  - Профильные системы и фурнитура
  - Сроки изготовления и монтажа
  - Гарантийные условия и сервис
  - Акции и спецпредложения
- Автоматическое уточняющее вопросы при неполном запросе
- Ссылки на подробные материалы и каталоги

### 3. 🗓️ Запись клиента на встречу
- Сбор необходимых данных: имя, телефон, адрес, предпочтительное время
- Проверка доступности слотов в реальном времени
- Автоматическое создание события в **Google Календаре**:
  - Название: «Замер окон — [Имя клиента]»
  - Описание: контакты, адрес, комментарий
  - Напоминания: за 24 часа и за 1 час до встречи
- Подтверждение записи клиенту в чате + опционально SMS/Email

### 4. 🔁 Дополнительные возможности
- Переключение на живого оператора при сложных запросах
- Логирование диалогов для анализа и дообучения модели
- Поддержка мультиканальности: сайт, Telegram, WhatsApp (через Suvvy)
